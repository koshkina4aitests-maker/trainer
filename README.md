# trainer

В проект добавлена новая реалистичная визуализация мышечной нагрузки для веб-интерфейса:

- анатомически более правдоподобная фигура (вид спереди и сзади),
- раздельные мышечные зоны по группам (грудь, плечи, руки, корпус, спина, ягодицы, ноги, голени),
- нейтральный body-tone и плавная premium-заливка нагрузки,
- интерактив по нажатию/тапу: название мышцы и текущая нагрузка,
- адаптивная вёрстка для мобильных устройств.

## Запуск

Откройте `index.html` в браузере или поднимите простой статический сервер из корня проекта:

```bash
python3 -m http.server 8080
```

Далее откройте `http://localhost:8080`.
